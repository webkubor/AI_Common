# Slack GIF Optimization Standard (Slack 动图规范)

> **目标**: 针对 Slack 平台进行尺寸与体积的最优适配。

## 1. 尺寸限制
*   **Emoji 专用**: 128x128 (硬性推荐)。
*   **消息流**: 480x480。

## 2. 性能参数
*   **帧率 (FPS)**: 10-20 FPS (平衡流畅度与体积)。
*   **时长**: 建议 3 秒以内（尤其是 Emoji）。
*   **色彩**: 限制在 48-128 色以压缩体积。

## 3. 动画技巧
*   **平滑度**: 使用 Easing 函数 (Ease-in-out, Bounce) 替代线性运动。
*   **循环**: 确保最后一帧与第一帧无缝衔接。
