# Persona System Standard (人像一致性规范 - Imagen 3 增强版)

> **目标**: 利用 Imagen 3 的语言理解能力，通过叙述性描述和多参考图锁死角色特征。

## 1. 骨相锁死 (Identity Hard-Lock)
*   **参考库路径**: `docs/ucd/persona_refs/` (包含底稿 `ref_01.png` 等)
*   **多图混合逻辑**: 
    *   执行时最多加载该目录下 5 张参考图。
    *   通过多角度（如正/侧位）参考图强化骨相锁死，降低单图生成的偶然误差。
*   **技术参数**: 
    *   `image_size`: 默认 `1K`，追求高保真时指定 `2K` 或 `4K` (大写 K)。
    *   `aspect_ratio`: 默认 `3:4` 或 `9:16` (适配移动端)。

## 2. 叙述性提示词技巧 (Narrative Prompting)
*   **禁用**: 严禁使用 "1girl, solo, long hair" 这种传统的关键词堆砌。
*   **启用**: 使用连贯的段落描述。
    *   *示例*: "一位有着温柔眼神和精致五官的年轻女性（基于参考图特征），她正坐在清晨阳光洒下的窗边看书。阳光透过发丝呈现出细腻的质感，皮肤白皙瓷感，穿着一件柔软的浅灰色羊绒毛衣。画面构图自然，带有电影质感。"

## 3. 审美与细节 (Aesthetic & Details)
*   **摄影术语**: 明确指定镜头（如 "85mm f/1.8"）、光影（如 "Golden hour rim light"）和景深。
*   **中式注入**: 保持冷白皮、胶原蛋白感和零毛孔的视觉要求。

## 4. 迭代逻辑
*   **利用思考模式**: 允许 AI 在生成前进行“构思推理”，优先输出构思草图进行确认（若工具支持）。
*   **多轮编辑**: 利用 `thought_signature` 保持上下文，在修改时直接描述“把毛衣换成蓝色”而不是重新描述整个场景。
