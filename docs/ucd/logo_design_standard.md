# Logo Design & Vector Standard (矢量设计规范)

> **目标**: 指导生成比例协调、现代简约的生产级 SVG Logo。

## 1. 几何与构图 (Geometry)
*   **比例**: 优先采用黄金比例 (1:1.618)。
*   **对齐**: 使用 4px/8px 网格对齐，严禁随意的小数坐标。
*   **负空间**: 鼓励利用镂空增加设计深度。

## 2. 技术规格 (Technical Specs)
*   **ViewBox**: 统一使用 `0 0 512 512`。
*   **透明度**: 默认不带背景色 (Transparent by Default)，除非明确要求 App Icon 模式。
*   **代码质量**: 语义化标签 (`<title>`, `<desc>`)，结构清晰的分组 (`<g>`)。

## 3. 配色风格
*   **质感**: 适度使用渐变 (Linear/Radial Gradient)，避免死板的纯色平铺。
*   **兼容性**: 必须考虑深色模式下的可见性。
