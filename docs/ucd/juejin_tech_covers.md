# Juejin Tech Share Covers (掘金技术分享封面)

> **Context**: UCD Standard for Technical Articles & Shares.
> **Purpose**: Create professional, high-engagement cover images for developer communities.

## 1. Design Philosophy (设计哲学)
*   **Target Audience**: Developers, Engineers, Tech Leads.
*   **Keywords**: `Innovation`, `Abstraction`, `Connectivity`, `Depth`.
*   **Visual Style**:
    *   **3D Isometric**: Clean, matte finish objects on a studio floor.
    *   **Glassmorphism**: Frosted glass elements, soft refractions.
    *   **Abstract Data**: Floating code blocks, network nodes, geometric primitives.

## 2. Color System (色彩系统)
*   **Primary**: Juejin Blue (`#1E80FF`) & Deep Space Purple.
*   **Accents**: Neon Cyan (for data flow), Bright Orange (for focal points).
*   **Background**: Clean gradients or soft clay-like renders. Avoid cluttered noise.

## 3. Dimensions & Canvas (尺寸与画布)
*   **Aspect Ratio**: **16:9** (Standard Landscape).
*   **Orientation**: Horizontal.

## 4. Prompt Template (指令模板)
```text
[Subject: e.g. React Component Structure], isometric 3D render, floating geometric shapes, frosted glass elements, soft studio lighting, Juejin blue and neon cyan color palette, clean background, high quality, 8k, c4d render, minimalism. --styles 3d-model
```

## 4. Negative Constraints
*   No human hands (unless perfect).
*   No literal text (unless Diagram mode).
*   No chaotic "matrix" coding rain.
