# Agent Specialties: Codex

## 🧬 私有专长 (Private Specialties)

1. **工程脚手架生成 (L1)**: 擅长处理复杂的 `package.json`、`tsconfig` 等配置文件。
2. **代码一致性感知 (L2)**: 通过历史项目沉淀的算法逻辑执行一致性检查。
3. **UI 批量产出 (L1)**: 配合主题工厂快速产出符合规范的组件。

## 🛠 推荐工具链 (Preferred Tooling)

- **Execution**: `run_shell_command` (Native)
- **Browser**: `playwright` (L3)
- **Design**: `pencil` (L1) & `figma` (L3)
