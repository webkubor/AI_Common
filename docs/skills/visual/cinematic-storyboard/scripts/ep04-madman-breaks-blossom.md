# 第四集：狂徒折花 (The Madman Breaks the Blossom)

> **场景**: 边境小镇，残阳下的破旧酒肆门口
> **时长**: 15秒
> **风格**: 写实武侠 (雪中悍刀行质感)

## A. 场景上下文
*   **氛围**: 尘土飞扬，杀气腾腾，残阳如血。
*   **音频**: 酒坛破碎声 -> 斩马刀划过地面的刺耳磨铁声 -> 师妹的闷哼。
*   **剧情**: 慕夕歌自诩无敌，撞见因喝酒没给钱被赶出的欧阳狂徒，言语冲突后，被欧阳狂徒随手一招击伤。
