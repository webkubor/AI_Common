# 电影级分镜与提示词生成器 (Cinematic Storyboard & Prompt Generator)

## 适用场景
当你想要将一个叙事想法转化为**生产级别的分镜脚本**和**AI视频提示词**时使用。本技能不限风格：支持2D动漫、3D动画（皮克斯/迪士尼风）、写实电影、赛博朋克、水彩等。

**触发词**: "脚本", "分镜", "视频提示词", "电影感拆解", "动画分镜", "帮我把这个故事动起来"。

## 核心工作流

### 1. 风格与设定分析
首先，根据老爹提供的信息确认以下内容（若未提供，则根据语境推断）：
*   **风格类型**: (例如：*新海诚风动漫、暗黑幻想3D、黑色电影、吉卜力风*)。
*   **场景/背景**: 环境设定 (例如：*雨中的赛博朋克城市、中世纪战场、温馨的乡间小屋*)。
*   **叙事弧 (Narrative Arc)**: 15秒内的微型故事线 (铺垫 -> 冲突 -> 高潮/定格)。
*   **音频氛围**: 听觉设计 (音乐类型、核心音效 SFX、配音基调)。

### 2. "动态叙事" 标准
无论何种风格，输出必须体现**动态感**：
*   **动漫 (Anime)**: 侧重 *速度线、冲击帧、眼部特写、斜角镜头 (Dutch Angle)*。
*   **3D动画**: 侧重 *挤压与拉伸、流体般的肢体动作、体积光、丰富的材质感*。
*   **电影 (Film)**: 侧重 *景深、跟焦切换、希区柯克变焦 (Dolly Zoom)、电影灯光、胶片颗粒*。

### 3. 输出格式
提供 **场景上下文 (Scene Context)** 标题，紧接着是 **分镜表格**，最后是 **综合提示词 (Synthesized Prompt)**。

#### A. 场景上下文 (Scene Context)
> **风格**: [例如：2D 手绘动漫]
> **场景**: [例如：日落时的屋顶]
> **音频**: [例如：忧郁钢琴 + 城市环境音]
> **故事**: [15秒剧情简述]

#### B. 分镜表格 (Storyboard Table)
| 时间 | 景别与镜头语言 | 动作与表演 | 视觉效果 (光影/色彩/特效) | 音频设计 (音效/音乐) |
| :--- | :--- | :--- | :--- | :--- |
| **00-04s** | **[景别]** + **[运镜]** | *发生了什么？主体动作？* | *灯光、微粒、色彩基调* | *具体音效与出现点* |

#### C. 综合提示词 (Synthesized Prompt - AI 视频专用)
一个高度描述性的段落，专为 Sora/Runway/Pika/Kling 等模型优化。**开头必须包含风格关键词。** 注意：此提示词必须使用**中文**编写。

## 示例：赛博朋克追逐 (动漫风格)

**输入**: "一个赛博朋克女孩在霓虹屋顶上躲避无人机追捕，夜晚，动漫风格。"

**输出**:

> **风格**: 90年代赛博朋克动漫 (攻壳机动队风格)
> **场景**: 新东京屋顶，大雨
> **音频**: 快节奏合成器贝斯 + 重呼吸声 + 雨声
> **故事**: 逃亡 (奔跑) -> 对峙 (被围) -> 信仰之跃 (跳跃)。

| 时间 | 景别与镜头语言 | 动作与表演 | 视觉效果 (光影/色彩/特效) | 音频设计 (音效/音乐) |
| :--- | :--- | :--- | :--- | :--- |
| **0-4s** | **低角度跟踪** (快速后退拉镜头) | 女孩向镜头冲刺，神色焦急，不断回头看。 | **霓虹粉/蓝** 在积水地面的反射。雨滴形成的速度线。 | 沉重的呼吸声 (贴耳)，溅起的水花声，远处的警报。 |
| **4-8s** | **高角度 / POV** (无人机视角) | 无人机群从上方俯冲，扫描灯晃动。女孩滑铲钻过管道。 | **探照灯** 刺穿雨幕。镜头光晕。阴暗、写实的质感。 | 高频机械嗡鸣声，电子扫描的嘀嗒声。 |
| **8-12s** | **极端特写** (双眼) | 女孩停下，双眼放大，瞳孔收缩。下定决心跳跃。 | 眼部 **赛博HUD界面** 覆盖。瞳孔中映出下方的城市。 | 音乐瞬间静止 (0.5s) -> 剧烈的心跳声。 |
| **12-15s** | **全景/远景** (慢动作) | 她纵身跃下楼顶进入深渊。无人机发射激光（擦肩而过）。 | **全城景观展现**。激光留下红色光迹。角色在巨大全息图前形成剪影。 | 合成器重音爆发 (Drop)，风声呼啸，激光炸裂声。 |

**综合提示词 (AI 专用)**:
> 90年代赛博朋克动漫风格，三渲二，高对比度。(0:00) 低角度跟踪镜头，赛博朋克女孩在雨后的霓虹屋顶向镜头狂奔，霓虹粉色和蓝色的灯光，神色焦急，带有速度线。(0:04) 切换至高角度无人机视角，成群的安全无人机用探照灯穿透暴雨，女孩滑行钻过障碍物。(0:08) 眼睛极端特写，赛博HUD界面覆盖，瞳孔中倒映着霓虹城市，眼神坚定。(0:12) 全景镜头，慢动作，她从大楼边缘跃入虚空，在巨大的全息广告前形成剪影，激光擦肩而过，留下红色光迹，极具氛围深度，4k分辨率。

## Agent 执行指令
1.  **识别** 用户想要的风格（动漫、写实、3D等）。若未指定，主动询问或根据描述推断。
2.  **设计** 背景环境和音频景观来支撑叙事。
3.  **编写** 表格，特别注意 **音频设计** 的层次感。
4.  **合成** 提示词，确保风格关键词位于显著位置。
