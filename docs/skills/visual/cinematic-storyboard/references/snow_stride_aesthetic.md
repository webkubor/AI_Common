# 雪中悍刀行叙事美学 (Sword Snow Stride Aesthetic)

> **核心原则**: 悍气、破碎、留白、江湖气。

## 1. 风雪意向 (The Blizzard)
*   **不仅仅是背景**: 风雪是角色。风雪的密度、速度应随剧情张力改变。
*   **物理反馈**: 头发上的冰霜、睫毛上的雪籽、靴子踏雪的深坑。

## 2. 暴力美学 (Violence & Impact)
*   **重剑无锋**: 攻击时强调力量感（雪花飞溅、地面裂开），而非轻飘飘的特效。
*   **动态模糊**: 快速动作时的电影级模糊，强调速度与力量的爆发。

## 3. 构图规范 (Composition)
*   **极端景别**: 
    *   **极端特写 (ECU)**: 只有双眼，映出杀机。
    *   **极端远景 (EWS)**: 茫茫雪原一人一剑，强调孤寂感。
*   **留白**: 画面中要有大量的空旷区域（大雪、天空），将主体挤压在角落，增加压迫感。

## 4. 角色符号
*   **大氅 (The Mantle)**: 身份与力量的象征，必须有惊人的体积感。
*   **断兵**: 碎掉的刀剑、被削去的发束，体现战斗的惨烈。
