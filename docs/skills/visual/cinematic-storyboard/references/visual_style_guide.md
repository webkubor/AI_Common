# 写实武侠视觉指南 (Realistic Wuxia Style Guide)

> **核心原则**: 物理真实 > 视觉华丽。拒绝页游感，拥抱电影感。

## 1. 光影协议 (Lighting)
*   **低调摄影 (Low-key Lighting)**: 强调高对比度，利用阴影隐藏细节，增加神秘感。
*   **自然光源**: 优先使用月光、火把、雪地反光、窗边侧光。
*   **体积光 (Volumetric Light)**: 雪中的丁达尔效应，光束必须能看见风雪的流动。

## 2. 色调与影调 (Color Grading)
*   **冷暖对比**: 整体基调偏冷（青、灰、蓝），但在角色面部或核心道具上保留温暖的肤色或火光。
*   **低饱和度**: 颜色要沉下去，强调布料、金属、木材的原始质感。

## 3. 材质表现 (Texturing)
*   **金属**: 刀剑要有细微的划痕、血槽、锻造纹理（不再是发亮的银色，而是暗沉的冷钢）。
*   **织物**: 丝绸的流动、粗麻的纹理、貂裘的根根分明，必须有物理重量感。
*   **皮肤**: 可见的毛孔、冻红的鼻尖、愈合的伤疤，拒绝塑料感。
