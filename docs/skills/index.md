# 🧩 技能矩阵 (Skills Matrix)

> **“CortexOS 是中枢神经，Skills 是可插拔的作战模组。”**

CortexOS 的能力被划分为两个层面：开源的**独立技能包**（推荐）与**本地集成能力**。

---

## 🌍 开源技能库 (Recommended)

所有关于账号运营、代码发版、内容分发的重型能力，均已拆分为独立的开源 Skill 包。它们拥有自己的版本号与独立的维护周期。

- [**查看开源技能总表 (GitHub)**](./github_repos.md)
  - 📦 **Omni-Publisher**: 多平台内容分发（小红书/公众号）。
  - 📦 **SCM-Ops**: 跨平台代码托管（GitHub/GitLab）。
  - 📦 **XHS-Manager**: 小红书多账户自动化管理。
  - ...以及更多（共 13+ 项）。

---

## 🏠 本地集成能力

这些能力深度耦合在 CortexOS 的内核中，主要负责大脑自身的运作、进化与元数据管理。

| 能力名称 | 说明 | 路径 |
| :--- | :--- | :--- |
| **Core 协议** | 定义大脑的基础通信与协作底座。 | [详情](./core/common_manifest.md) |
| **项目进化 SOP** | 全量更名、品牌重构与 UI 动效注入流程。 | [详情](./core/project_evolution_skill.md) |
| **碎片知识管家** | 负责代码片段（Snippets）的快速存取。 | [详情](./knowledge/snippet_master.md) |
| **深度思考模式** | 激活 Socratic 逻辑推理与多维评估。 | [详情](./core/think-skill.md) |
