# Preferred Icon Libraries (视觉资产准则)

> **Source**: 老爹 (webkubor) 2026-02-06
> **Usage**: 寻找 App 图标时必须优先遵循此序列。

## 1. Super Tiny Icons (优先级: High)
- **URL**: https://www.supertinyicons.org/
- **特点**: 极致优化，适合标准品牌 Logo。

## 2. Streamline (优先级: Medium)
- **URL**: https://www.streamlinehq.com/
- **特点**: 搜索功能强，适合寻找小红书等特定应用图标。

## 3. SVG Icons (优先级: Fallback)
- **URL**: https://svgicons.com/
- **特点**: 海量补充。
