## [2026-01-20] Satori
Vercel Satori 是一个将 HTML 和 CSS 转换为 SVG 的库。
关键词：Satori、Vercel、Open Graph、OG Image、社交分享图、SVG、JSX、React、HTML/CSS 转 SVG

核心要点：
- 主要用途：动态生成 Open Graph (OG) 图片，可用 React/JSX 组件化方式设计图片。
- 支持 JSX：可直接使用 JSX 定义布局与内容。
- Flexbox 布局：支持 CSS Flexbox，便于排版。
- 跨平台：可在浏览器、Node.js、Web Workers 中运行。
- 字体处理：负责字体渲染并嵌入 SVG，保证跨设备一致。
- 限制：只支持部分 HTML 元素与 CSS 属性，不是完整浏览器引擎。
> 适合用代码（尤其 React/JSX）动态生成高质量图片的场景。
